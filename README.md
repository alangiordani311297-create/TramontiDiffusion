# Tramonti Diffusion | Luxury Dealership Platform

An ultra-premium automotive dealership platform built with Next.js 15, Tailwind CSS v4, and Google Gemini AI.

## 🌟 Features

- **Luxury Aesthetic**: Custom "Velluto" design system with gold accents and dark mode.
- **Interactive Showroom**: Filterable inventory of Supercars, Hypercars, and Classics.
- **AI Concierge**: Real-time sales assistant powered by Google Gemini 2.0.
- **Immersive Experience**: Video backgrounds, infinite carousels, and smooth animations.
- **Admin Dashboard**: Dedicated area for inventory management (Route: `/admin`).

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4 (CSS Variables)
- **Icons**: Lucide React
- **AI**: Google GenAI SDK
- **Language**: TypeScript

## 🚀 Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

- `app/(public)`: Public facing landing page and subpages.
- `app/(admin)`: Restricted admin area.
- `components/`: Reusable UI components (Hero, Navbar, Concierge, etc.).
- `services/`: External integrations (Gemini AI).
- `constants.ts`: Mock data and configuration.
